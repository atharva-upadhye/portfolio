# Portfolio website
This repository contains source code for my portfolio website.
## Tech stack
- language: Typescript
- framework: NextJS
  - parallel routes
- custom hooks:
  - useTheme
  - useShowScrollToTop
- custom components:
  - Theme Switcher
- CSS utility:
  - TailwindCSS
## Features
- All custom made components, no use of any 3rd party component library
- A portolio with perfect Lighthouse score!
